# Munazzam-App-BE

Munazzam backend API — Express + TypeScript + Supabase.

## Setup

```bash
npm install
cp .env.example .env
# fill in Supabase and JWT secrets
npm run dev
```

## Scripts

- `npm run dev` — local API server (watch mode)
- `npm run build` — compile TypeScript
- `npm start` — run compiled server
- `npm run typecheck` — type check only
